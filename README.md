
# Product Explorer Dashboard

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Features
- Product listing from FakeStore API
- Search & category filter
- Favorites (localStorage)
- Product detail pages
- Responsive UI

## Setup
```bash
npm install
npm run devnotepad README.md

```

## Assumptions
- Client-side state for simplicity
- No external UI libraries
